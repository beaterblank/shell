# shell
bash like shell implementation

## example of commands are :

 * echo string
 * add 1 2 - adds the 2 numbers
 * sub 2 3 - subsracts the 2 numbers
 * mul 1 4  - multiplies the 2 numbers
 * div 4 5  - divides the 2 numbers
 * touch fileName - creates a file
 * cat fileName - reads a file
 * cd directory - change directory
 * cd ..  - change directory
 * rm fileName  - removes a file
 * copy fileName - copies a file
 * pwd  - prints present working directory
 * ls - lists all files in curerent working directory
 * date - prints todays data
 * cal  - prints the calandar for the month
 * mkdir  - creates a directory
 * rmdir  - removes a directory
 * df - shows the memory usage
 * ps - shows the current processes running
 * kill pid - kills the process with the pid
